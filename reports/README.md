# Reports: Reading Guide

This folder preserves the evidence trail without asking a visitor to infer
which document matters first. The website is the guided entry point; these
files are the inspectable record behind it.

## Suggested order

1. **Start with the submitted work.** The exact original course report is
   published at `public/report/original-course-report.pdf`. It is historical
   coursework by Sahil Mhatre, Ali Hasan, and Corey Chen, not silently
   upgraded into a later claim.
2. **Read the independent extension report.**
   `public/report/representation-drift-lab-report.pdf` explains the rebuilt
   pipeline, corrected experiment design, findings, contradictions, and
   limitations in one document.
3. **Inspect the current evidence status.**
   `expanded-local-preregistration.md` records the completed expanded local
   check; `full-scale-preregistration.md` locks the next unexecuted GPU tier.
4. **Inspect boundaries when needed.**
   `fairness-feedback-loop-scope-audit.md` states what this benchmark can and
   cannot say about human impact. `excluded-runs.md` preserves a failed
   validity check rather than hiding it.
